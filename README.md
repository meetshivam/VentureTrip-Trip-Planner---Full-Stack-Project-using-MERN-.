# VentureTrip-Trip-Planner---Full-Stack-Project-using-MERN-.
Venture is a sleek MERN-stack Trip Planner to manage global itineraries. Integrated with Open-Meteo API for real-time weather forecasts. Built with React, Node, and Tailwind CSS; uses local MongoDB for data persistence. Organize your trips, milestones, and packing lists in one modern, responsive dashboard.
